HTML stands for "Hypertext Markup Language." This is the basis for every web page. HTML documents contain text content (what you see), markup (what it looks like), references to other documents, and links to other pages.This video is a good place to start.

How the Web Works

First, let's define the major pieces of the web:

-Servers - Computers that host the files that make up the web
-Internet - world's largest computer nerwork
-Browser - program that displays files on your computer from the web
-HTTP - the main protocol used by browsers to communicate with servers

The browser on your computer and makes requests to servers via the internet using the HTTP protocol. The servers respond with files that your browser displays. Here's a good explanation.

HTML Tags / Elements

HTML markup is made up "tags." Tags look something like this: <"Tag name"> Contents <"/Tag name">

The two tag names are always the same, while the first one is called the opening tag and the second is called the closing tag. The opening tag, its contents, and the closing tag as a whole can be referred to as an element. HTML tags are used to help the browser identify the type of each element.You can find an introduction of the two here.

Computers Are Stupid?

It's important to remember that computers interpret the instructions humans give them literally. One small typo or mistake that "smart" human could correct can lead to huge problems when given to a computer. What do you think happens when you forget to insert a closing tag? Find out here.

Inline vs. Block Elements

You may notice that text on multiple lines will appear on a single line when written in HTML. Tags such as <""br""> and <"p"> can aid in creating line breaks. The break tag is an inline element that signifies the end of a line. The paragraph tag is a block element that creates an invisible box around its contents. This video is a big help.
